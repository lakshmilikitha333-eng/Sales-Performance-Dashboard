# Sales Performance Dashboard

## Project Overview
This project analyzes retail sales data using SQL, Microsoft Power BI, and DAX to identify sales trends, profitability, regional performance, customer purchasing behavior, and payment preferences.

## Business Objective
Analyze retail sales performance to identify trends, evaluate profitability, and support data-driven business decisions.

## Dataset
Public retail sales transaction dataset containing product, customer, regional, payment, and order information.

## Tools & Technologies
- SQL
- Microsoft Power BI
- DAX

## Key Performance Indicators (KPIs)
- Total Sales
- Profit
- Total Orders
- Average Order Value (AOV)
- Profit Margin (%)
- Customer Rating

## Dashboard Features
- Monthly Sales Trend
- Sales by Product Category
- Top Selling Products
- Sales by Region
- Payment Method Distribution
- Profit by Product Category

## Business Recommendations
- Increase inventory and promotional campaigns for high-performing product categories.
- Strengthen marketing initiatives in underperforming regions.
- Increase Average Order Value (AOV) through cross-selling and product bundles.
- Optimize pricing and supplier costs to improve profit margins.
- Encourage digital payment methods using targeted promotional offers.
- Use monthly sales trends for better demand forecasting and inventory planning.

## Outcome
Developed an interactive Power BI dashboard that transformed raw retail sales data into actionable business insights, enabling informed decision-making through KPI monitoring, sales trend analysis, regional performance evaluation, and profitability analysis.

## Skills Demonstrated
SQL • Data Cleaning • Data Transformation • DAX • Power BI • Data Visualization • Business Analysis
